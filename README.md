#  ClassRoomPB

Sistema de controle acadêmico simplificado desenvolvido em Java para a disciplina de Engenharia de Software 2 (2026.1).

##  Sobre o projeto

O ClassRoomPB é um sistema CLI (Command Line Interface) que gerencia o ciclo acadêmico completo de uma instituição de ensino, contemplando quatro perfis de usuário: **Aluno**, **Professor**, **Coordenador** e **Administrador**.

##  Funcionalidades

- Cadastro e autenticação de usuários por perfil
- Gestão de cursos, disciplinas e períodos letivos
- Oferta e gerenciamento de turmas
- Matrícula com validação de pré-requisitos, vagas e conflito de horários
- Lista de espera automática
- Registro de frequência e lançamento de notas
- Cálculo automático de médias e situação final do aluno
- Histórico acadêmico
- Relatórios gerenciais

##  Tecnologias

- **Java** (sem frameworks externos)
- **JUnit** para testes unitários 
- **Padrões de Projeto** (Design Patterns)
- Persistência local em disco
